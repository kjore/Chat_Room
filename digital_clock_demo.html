Digital Clock Features:

  • ✅ Real-time updates every second
  • ✅ Multiple timezone support (Local, UTC, EST, PST)
  • ✅ Clean UI using existing AppTheme colors and fonts
  • ✅ Integrated into HomePageFrame at the bottom
  • ✅ Monospaced font for consistent time display
  • ✅ Timer cleanup on window close
  • ✅ Minimal integration with existing codebase
